# 10_Learning — 技術學習筆記

> tags: #Meta

## 目的

記錄課外的技術學習內容，包含 YouTube 影片筆記、線上課程心得、技術文章筆記等。

> 💡 AI 領域發展快速，YouTube 是最即時的學習管道，優先於書籍。

## 子目錄

| 目錄 | 說明 | 命名格式 |
|---|---|---|
| `YouTube/` | YouTube 影片 / 頻道學習筆記 | `頻道名_影片主題.md` |
| `Online_Courses/` | 線上課程筆記 | `平台_課程名稱.md` |
| `Articles/` | 技術文章筆記 | `YYYY-MM-DD_標題.md` |
| `Books/` | 書籍閱讀摘要（選讀） | `書名.md` |

## 範例

```
10_Learning/
├── YouTube/
│   └── 3Blue1Brown_Attention機制視覺化.md
│   └── Andrej_Karpathy_GPT_from_Scratch.md
├── Online_Courses/
│   └── Coursera_DeepLearning_Specialization.md
├── Articles/
│   └── 2026-03-15_RAG_Best_Practices.md
└── Books/
    └── Designing_Machine_Learning_Systems.md
```

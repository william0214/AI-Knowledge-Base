# 10_Learning — 技術學習筆記

> tags: #Meta

## 目的

記錄課外的技術學習內容，包含書籍摘要、線上課程心得、技術文章筆記等。

## 子目錄

| 目錄 | 說明 | 命名格式 |
|---|---|---|
| `Books/` | 書籍閱讀摘要 | `書名.md` |
| `Online_Courses/` | 線上課程筆記 | `平台_課程名稱.md` |
| `Articles/` | 技術文章筆記 | `YYYY-MM-DD_標題.md` |

## 範例

```
10_Learning/
├── Books/
│   └── Designing_Machine_Learning_Systems.md
├── Online_Courses/
│   └── Coursera_DeepLearning_Specialization.md
└── Articles/
    └── 2026-03-15_RAG_Best_Practices.md
```

## 3.3 用 JSON 結構化資料建立訓練集

### 💡 思考
- 為什麼 `ld+json` 能降低爬蟲欄位抽取成本？
- 在實務上，欄位結構穩定和資料量，哪一個更重要？

## 3.4 文本清洗與前處理原則

### 💡 思考
- 中文文本清理與英文相比，最大的差異點在哪裡？
- 情感分析任務中，過度清洗可能會破壞哪些訊號？

### 📌 重點
- 前處理的基本目標：
    - 去除 HTML 與雜訊。
    - 保留中文、英數與必要符號。
    - 正規化空白與欄位格式。
- 這一層品質會直接影響分詞、TF-IDF、情感分析與後續微調效果。

### 🔧 實作
```python
# 來源：依課程脈絡整理
# 用途：中文文本清洗範例

import re

def clean_text(text: str) -> str:
        text = re.sub(r"<[^>]+>", "", text)
        text = re.sub(r"[^\w\s\u4e00-\u9fff]", "", text)
        text = re.sub(r"\s+", " ", text).strip()
        return text
```

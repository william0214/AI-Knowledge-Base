## 5.4 DavidLanz 中文情感模型 Lab

### 💡 思考
- 直接復用已微調模型，對企業 PoC 的價值是什麼？
- 五分類比三分類多出的訊息，在哪些商業場景最有用？

### 📌 重點
- 課堂示例模型：`DavidLanz/fine_tune_chinese_sentiment`
- 標籤共五類：
	- `0: Semi-negation`
	- `1: Negation`
	- `2: Neutral`
	- `3: Semi-positive`
	- `4: Positive`
- 這代表課堂不只談二元或三元情感，也涵蓋較細粒度的情緒區分。

### 🔧 實作
```python
# 來源：課程講義 PDF 第 38-39 頁
# 用途：載入 DavidLanz 中文情感模型

from transformers import BertForSequenceClassification, BertTokenizer

tokenizer = BertTokenizer.from_pretrained("DavidLanz/fine_tune_chinese_sentiment")
model = BertForSequenceClassification.from_pretrained("DavidLanz/fine_tune_chinese_sentiment")

label_map = {
		0: "Semi-negation",
		1: "Negation",
		2: "Neutral",
		3: "Semi-positive",
		4: "Positive",
}
```

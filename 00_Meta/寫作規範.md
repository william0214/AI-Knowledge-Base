# 寫作規範

> tags: #Meta

## Markdown 格式

| 層級 | 用途 | 範例 |
|---|---|---|
| `# H1` | 文件標題（每個檔案僅一個） | `# 情感分析` |
| `## H2` | 大章節 | `## 模型訓練` |
| `### H3` | 子章節 / 功能區塊 | `### 💡 思考` |
| `#### H4` | 細項 | `#### 參數說明` |
| `> blockquote` | 標籤、引用、重要提醒 | `> tags: #NLP #Day1` |

## 課程筆記模板結構

每個 H2 章節下包含以下四個標準區塊：

```markdown
## 章節名稱

### 💡 思考
- 問題 1？
- 問題 2？

### 📌 重點
- 

### 🔧 實作
\```python
# 程式碼
\```

### 📚 延伸
- [資源名稱](連結)
```

## 程式碼區塊規範

- 標注語言：\`\`\`python
- 附上簡短註解說明用途
- 標注來源：`# 來源：老師範例 / 自己實作 / 官方文件`

```python
# 來源：課程範例
# 用途：使用 Jieba 進行中文斷詞
import jieba
words = jieba.cut("自然語言處理很有趣")
print(" / ".join(words))
```

## 圖表規範

- 圖片存放於 `07_Diagrams/`
- 使用 Mermaid 語法（可在 Markdown 中直接渲染）或圖片檔
- 圖片命名：`主題_描述.png`
- 引用格式：`![描述](../07_Diagrams/主題_描述.png)`

## 引用規範

- 標注出處（課程講義、官方文件、論文）
- 格式：`> 來源：[名稱](連結)`
- 老師口述的重點用 `> 🗣️ 老師提到：...` 標記

## 外部連結格式

```markdown
- [HuggingFace Transformers](https://huggingface.co/docs/transformers/)
- [LangChain 官方文件](https://python.langchain.com/)
```

# 13_Insights — 產業趨勢與技術觀點

> tags: #Meta

## 目的

追蹤 AI/NLP 產業動態、記錄技術研討會心得、整理個人技術觀點與反思。

## 子目錄

| 目錄 | 說明 | 命名格式 |
|---|---|---|
| `Trends/` | 產業趨勢分析 | `YYYY-MM_主題.md` |
| `Conferences/` | 研討會 / 社群活動筆記 | `YYYY-MM-DD_活動名稱.md` |
| `Tech_Notes/` | 技術觀點與反思 | `主題.md` |

## 範例

```
13_Insights/
├── Trends/
│   ├── 2026-03_AI_Agent_框架_趨勢.md
```

# 12_SOP — 個人 SOP 與工具備忘

> tags: #Meta

## 目的

記錄個人工作流程、開發環境設定、工具使用備忘，確保可重現且不遺忘關鍵步驟。

## 子目錄

| 目錄 | 說明 | 命名格式 |
|---|---|---|
| `Tools/` | 工具使用備忘 | `工具名稱.md` |
| `Environments/` | 開發環境設定 | `環境描述.md` |
| `Workflows/` | 工作流程 SOP | `流程名稱.md` |

## 範例

```
12_SOP/
├── Tools/
│   ├── Git_常用指令.md
│   ├── Docker_備忘錄.md
│   └── VS_Code_快捷鍵.md
├── Environments/
│   ├── Python_venv_設定.md
│   ├── Azure_DevOps_CI_CD.md
│   └── Mac_開發環境_初始化.md
└── Workflows/
    ├── 外訓心得報告_撰寫流程.md
    ├── 模型部署_標準流程.md
    └── Code_Review_Checklist.md
```
